# React 19 Advanced Features

Welcome to the React 19 Advanced Features Guide! React 19 introduces a paradigm shift in how we manage state, perform data fetching, handle forms, and optimize performance. This module covers everything you need to know about the latest features and architectural patterns.

## Topics Covered

1. [React 19 New Features](./01-react-19-new-features.md)
   - Actions, `useActionState`, and Form Actions
   - The `useOptimistic` hook
   - The `use()` API for reading resources
   - Document metadata and asset loading
   - `ref` as a prop and `Context` as a provider

2. [Server Components Deep Dive](./02-server-components-deep-dive.md)
   - React Server Components (RSC) vs Client Components
   - Server Actions and Mutations
   - Data fetching in Server Components
   - Streaming and Suspense

3. [Advanced Concurrent Features](./03-advanced-concurrent-features.md)
   - Async Transitions and `useTransition`
   - Suspense boundaries and Error Boundaries
   - Streaming Server-Side Rendering (SSR)
   - Selective Hydration

4. [Performance Optimization](./04-performance-optimization.md)
   - React Compiler (Auto-Memoization)
   - When manual optimization (`useMemo`/`useCallback`) is needed
   - Profiling and Code Splitting

5. [Migration Guide](./05-migration-guide.md)
   - Upgrading from React 18 to 19
   - Breaking changes, deprecations, and codemods
   - Testing strategy during migration

## Prerequisites

- Solid understanding of React 18 functional components and hooks
- Proficiency in TypeScript
- Understanding of Server-Side Rendering concepts

Let's dive in!
