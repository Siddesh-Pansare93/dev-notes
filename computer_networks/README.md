## Prerequisites

### Required Knowledge
- Basic understanding of computers and operating systems
- Familiarity with command line (Linux/Windows terminal)
- Basic understanding of binary and hexadecimal numbers

### Recommended Tools
- **Network Tools**: ping, traceroute, nslookup, netstat
- **Packet Analyzer**: Wireshark
- **Network Simulator**: Cisco Packet Tracer or GNS3 (optional)
- **Virtual Machines**: VirtualBox or VMware (for lab setup)
- **Operating Systems**: Linux (Ubuntu/CentOS) and Windows

### Programming (Optional but Recommended)
- Python for network scripting (using `socket`, `scapy`, `requests` libraries)
- Bash scripting for automation

## Key Concepts Covered

### Network Models
| Concept | Description | Real-world Example |
|---------|-------------|-------------------|
| OSI Model | 7-layer conceptual framework | Understanding how data travels from application to wire |
| TCP/IP Model | 4-layer practical implementation | Internet protocol suite |
| Client-Server | Request-response architecture | Web browsing, email |
| Peer-to-Peer | Distributed networking | BitTorrent, blockchain |

### Addressing and Routing
| Concept | Description | Use Case |
|---------|-------------|----------|
| IPv4 | 32-bit addressing (4.3B addresses) | Most current internet infrastructure |
| IPv6 | 128-bit addressing (340 undecillion addresses) | Future-proof internet addressing |
| Subnetting | Dividing networks into smaller segments | Network organization, security |
| CIDR | Classless addressing notation | Efficient IP allocation |
| Routing | Path determination for packets | Internet backbone, enterprise networks |

### Protocols
| Protocol | Layer | Purpose | Port |
|----------|-------|---------|------|
| HTTP/HTTPS | Application | Web communication | 80/443 |
| DNS | Application | Domain name resolution | 53 |
| FTP | Application | File transfer | 20/21 |
| SMTP | Application | Email sending | 25 |
| TCP | Transport | Reliable connection | - |
| UDP | Transport | Fast, connectionless | - |
| IP | Network | Packet routing | - |
| ICMP | Network | Error reporting, diagnostics | - |

### Security Concepts
| Concept | Purpose | Technology |
|---------|---------|-----------|
| Encryption | Data confidentiality | AES, RSA |
| Authentication | Identity verification | Certificates, passwords |
| Firewall | Access control | iptables, pfSense |
| VPN | Secure remote access | OpenVPN, IPSec |
| IDS/IPS | Threat detection | Snort, Suricata |
