## Key Concepts Covered

### Operating System Responsibilities

| Responsibility | Description | Example |
|----------------|-------------|---------|
| **Process Management** | Creating, scheduling, and terminating processes | Running multiple applications simultaneously |
| **Memory Management** | Allocating and deallocating memory | Virtual memory, paging |
| **File System Management** | Organizing and storing files | Creating, reading, writing files |
| **I/O Management** | Managing input/output operations | Keyboard input, disk reads |
| **Security** | Protecting system resources | User authentication, file permissions |
| **Networking** | Communication between systems | TCP/IP stack implementation |

### Process States

| State | Description | Transitions |
|-------|-------------|-------------|
| **New** | Process being created | → Ready |
| **Ready** | Ready to execute, waiting for CPU | → Running |
| **Running** | Executing on CPU | → Ready, Waiting, Terminated |
| **Waiting** | Waiting for I/O or event | → Ready |
| **Terminated** | Finished execution | None |

### Memory Management Techniques

| Technique | Description | Pros/Cons |
|-----------|-------------|-----------|
| **Contiguous Allocation** | Process occupies single memory block | Simple / External fragmentation |
| **Paging** | Fixed-size memory blocks | No external fragmentation / Internal fragmentation |
| **Segmentation** | Variable-size logical units | Logical organization / External fragmentation |
| **Virtual Memory** | Illusion of unlimited memory | Large address space / Page faults |

### File System Types

| File System | OS | Features |
|-------------|----|---------| 
| **ext4** | Linux | Journaling, large files, backward compatible |
| **NTFS** | Windows | Journaling, security features, compression |
| **APFS** | macOS/iOS | Optimized for SSDs, snapshots, encryption |
| **Btrfs** | Linux | Copy-on-write, snapshots, RAID support |
| **ZFS** | Unix-like | Data integrity, compression, deduplication |

## Hands-on Projects

Throughout this guide, you'll work on practical projects:

1. **Custom Shell**: Build a simple command-line shell
2. **Process Scheduler**: Implement scheduling algorithms (FCFS, SJF, Round Robin)
3. **Memory Allocator**: Create a custom malloc/free implementation
4. **Simple File System**: Implement basic file system operations
5. **Thread Synchronization**: Solve producer-consumer problem
6. **Page Replacement Simulator**: Implement and compare algorithms (LRU, FIFO, Optimal)
7. **System Monitor**: Build a tool to display process and system information
8. **Mini OS Kernel**: Create a basic kernel that boots and prints to screen
