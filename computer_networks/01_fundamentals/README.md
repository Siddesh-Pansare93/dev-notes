# 01. Fundamentals

This section covers the foundational concepts of computer networking. You'll learn about network basics, the OSI and TCP/IP models, network topologies, devices, and how data is transmitted across networks.

## Topics Covered

1. **[Introduction to Computer Networks](./01_introduction_to_networks.md)**
   - What is a computer network?
   - Types of networks (LAN, WAN, MAN, PAN)
   - Network architecture and components
   - Brief history of the internet

2. **[OSI Model](./02_osi_model.md)**
   - The 7 layers of the OSI model
   - Function of each layer
   - Data encapsulation and de-encapsulation
   - Layer interactions

3. **[TCP/IP Model](./03_tcp_ip_model.md)**
   - The 4 layers of TCP/IP
   - Comparison with OSI model
   - Internet protocol suite
   - Common protocols at each layer

4. **[Network Topologies and Architectures](./04_network_topologies.md)**
   - Physical vs. logical topologies
   - Star, bus, ring, mesh, hybrid topologies
   - Client-server vs. peer-to-peer
   - Network design considerations

5. **[Network Devices](./05_network_devices.md)**
   - Hubs, switches, and routers
   - Bridges and gateways
   - Modems and access points
   - Layer 2 vs. Layer 3 devices

6. **[Data Transmission and Encoding](./06_data_transmission.md)**
   - Analog vs. digital transmission
   - Bandwidth and throughput
   - Multiplexing techniques
   - Error detection and correction

## Learning Objectives

By the end of this section, you will:

- Understand what computer networks are and their classifications
- Master the OSI and TCP/IP reference models
- Identify different network topologies and their use cases
- Recognize common network devices and their functions
- Understand how data is transmitted and encoded

## Prerequisites

- Basic computer literacy
- Familiarity with basic computing concepts
- No prior networking knowledge required

## Estimated Time

**2-3 hours** to complete all tutorials in this section.
