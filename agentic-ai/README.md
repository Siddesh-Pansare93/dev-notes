# Agentic AI — LangChain & LangGraph, 0 se 100 tak

AI agents banana seekhna hai — simple chatbot se lekar production mein chalne wale multi-agent systems tak? Yeh section bilkul zero se shuru hota hai aur tumhe production-grade agents deploy karne tak le jaata hai. Poora content **LangChain** aur **LangGraph** pe based hai, aur do parallel tracks mein available hai:

- **[Python Track](./python/README.md)** — LangChain + LangGraph (Python SDKs), FastAPI ke saath
- **[JavaScript/TypeScript Track](./javascript/README.md)** — LangChain.js + LangGraph.js, Express/Next.js ke saath

Dono tracks bilkul same 24-chapter curriculum follow karte hain — bas language aur framework alag hai. Agar tum Node.js dev ho, JavaScript track se shuru karo. Agar Python seekh rahe ho ya seekh chuke ho, Python track use karo.

## Kya seekhoge

**Part 1 — Foundations (Chapters 1-11):** LLMs ko call karna, prompts, structured output, chains (LCEL), memory, tools/function calling, apna pehla agent banana, RAG, observability, aur ek chain/agent ko API server ke peeche deploy karna.

**Part 2 — LangGraph (Chapters 12-21):** Graph-based agents — state graphs, conditional routing, state management, human-in-the-loop approvals, subgraphs, multi-agent systems, streaming, aur LangGraph ko API server pe run karna.

**Part 3 — Production (Chapters 22-24):** Advanced architectures (planner-executor, reflection), testing non-deterministic agents, aur real production concerns — cost control, guardrails, monitoring, scaling.

Har chapter mein working code hai, Hinglish explanation hai, aur real-world Indian analogies hain jo concepts ko turant click karwa dete hain.
