# CI/CD - Continuous Integration & Deployment

Learn to build automated pipelines with GitHub Actions.

## Topics Covered

1. **CI/CD Concepts** - CI vs CD, pipelines, automated testing
2. **GitHub Actions Basics** - Workflows, jobs, steps, triggers
3. **Building & Testing** - Automated builds, test runners, code quality
4. **Docker Image CI/CD** - Build, tag, push to registries (Docker Hub, ECR)
5. **Deployment Strategies** - Blue-green, canary, rolling deployments
6. **Secrets Management** - GitHub Secrets, environment variables, security
7. **Advanced Workflows** - Matrix builds, reusable workflows, caching

## Prerequisites

- GitHub account
- Basic Git knowledge
- Understanding of Docker (complete 01_fundamentals first)

## What You'll Build

By the end of this section, you'll be able to:
- Create CI/CD pipelines with GitHub Actions
- Automate testing and code quality checks
- Build and push Docker images automatically
- Deploy applications to various environments
