# 01. Fundamentals

This section covers the foundational concepts of operating systems. You'll learn what operating systems are, their architecture, how they interact with hardware and software, different types of OS, kernel designs, and the boot process.

## Topics Covered

1. **[Introduction to Operating Systems](./01_introduction_to_os.md)**
   - What is an operating system?
   - OS responsibilities and services
   - History and evolution
   - Types of operating systems

2. **[OS Architecture and Structure](./02_os_architecture.md)**
   - Monolithic vs microkernel architecture
   - Layered approach
   - Hybrid systems
   - Operating system components

3. **[System Calls and APIs](./03_system_calls.md)**
   - What are system calls?
   - Types of system calls
   - User mode vs kernel mode
   - System call implementation

4. **[Operating System Types](./04_os_types.md)**
   - Batch, time-sharing, real-time
   - Distributed and network OS
   - Mobile and embedded OS
   - Cloud and virtualized OS

5. **[Kernel Architecture](./05_kernel_architecture.md)**
   - Monolithic kernel
   - Microkernel
   - Hybrid kernel
   - Exokernel and unikernel

6. **[Boot Process and System Initialization](./06_boot_process.md)**
   - BIOS/UEFI
   - Bootloader (GRUB)
   - Kernel loading
   - Init system (systemd, SysV)

## Learning Objectives

By the end of this section, you will:

- Understand the role and purpose of operating systems
- Identify different OS architectures and their trade-offs
- Explain how applications interact with the OS through system calls
- Classify different types of operating systems
- Understand kernel designs and their implementations
- Trace the boot process from power-on to user login

## Prerequisites

- Basic programming knowledge (C or Python)
- Understanding of computer hardware components
- Familiarity with command line

## Estimated Time

**3-4 hours** to complete all tutorials in this section.
